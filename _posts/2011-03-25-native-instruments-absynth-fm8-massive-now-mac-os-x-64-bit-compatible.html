--- layout: post title: Native Instruments Absynth, FM8, Massive Now Mac OS X 64-Bit Compatible published: true ---

Native Instruments has updated Massive,FM8 and Absynth with Mac OS X 64-bit compatibility (AU, VST & Standalone).

64-bit support will provide performance benefits for users with more than 4 GB of RAM on their systems.

This is a free update. Check the NI Updates page.

Here’s what’s new in each app:

Absynth V5.0.3

  • Mac OS X 64-bit compatibility (AU, VST & Standalone).
  • Modified template folder location.
  • Morph Edit page crash fixed.
  • Program Change crash fixed.
  • Renaming Macro Controls fixed.
  • Several graphic glitches and lots of smaller bugs fixed.

FM8 V1.1.1

  • Mac OS X 64-bit compatibility (AU, VST & Standalone).
  • Several smaller bug fixes.

Massive V1.2.1

  • Mac OS X 64-bit compatibility (AU, VST & Standalone).
  • CoreAudioDevice crash fixed.
  • Several smaller bug fixes.
